# 🚂 Erika's Trains - Calcolo Biglietto Treno

Un'applicazione web interattiva per calcolare il prezzo del biglietto ferroviario con sconti personalizzati in base all'età del passeggero.

## 🎯 Descrizione

Questo progetto è un esercizio di sviluppo front-end che consente agli utenti di:
- Inserire dati personali (nome, età, chilometri da percorrere)
- Calcolare automaticamente il prezzo del biglietto in base a tariffe ridotte
- Visualizzare il riepilogo del biglietto in tempo reale

## ✨ Features

- **Validazione Form Personalizzata**: Controlli lato client con messaggi d'errore in italiano
- **Sistema di Sconti Automatici**:
  - Passeggeri under 18: sconto 20%
  - Passeggeri over 65: sconto 40%
  - Tariffà standard per altri
- **Design Responsive**: Ottimizzato per desktop, tablet e mobile
- **UI Moderna**: Palette colori coerente con Bootstrap 5.3.3
- **Feedback Visivo**: Validazione degli input con stili Bootstrap

## 🛠️ Tecnologie Utilizzate

- **HTML5**: Struttura semantica
- **CSS3**: Styling con Bootstrap 5.3.3 e personalizzazioni custom
- **JavaScript**: Logica applicativa, validazione e manipolazione del DOM
- **Bootstrap Icons**: Icone SVG responsive
